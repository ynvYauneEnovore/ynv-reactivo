# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh


Vite: Vite es una herramienta de construcción y servidor de desarrollo que tiene como objetivo proporcionar flujos de trabajo de desarrollo rápidos y eficientes para proyectos web modernos.

React: React es una biblioteca de JavaScript para construir interfaces de usuario. Se utiliza ampliamente para crear aplicaciones web dinámicas e interactivas.

HMR (Reemplazo de Módulo en Caliente): El HMR es una función que te permite ver los cambios que realizas en tu código reflejados de inmediato en el navegador sin tener que actualizar la página manualmente. Esto agiliza significativamente el proceso de desarrollo.

ESLint: ESLint es un linter de JavaScript popular que te ayuda a mantener una calidad de código consistente y a adherir a estándares de codificación.

@vitejs/plugin-react: Este complemento proporciona la integración entre Vite y React, habilitando "Fast Refresh" mediante el compilador Babel.

@vitejs/plugin-react-swc: Este complemento alternativo brinda integración con SWC, un compilador JavaScript/TypeScript muy rápido. Es otra opción para lograr "Fast Refresh" en un proyecto Vite.
